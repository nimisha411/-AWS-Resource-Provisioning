# 🚀 AWS Resource Provisioning using Python boto3

## 📌 Project Overview
This project automates AWS infrastructure provisioning using Python and boto3.

The script automatically:
- Creates EC2 instances
- Creates S3 buckets
- Uses IAM permissions for secure access

---

## 🧰 AWS Services Used
- Amazon EC2
- Amazon S3
- AWS IAM
- boto3 SDK

---

## 💻 Technologies Used
- Python
- boto3
- AWS CLI
